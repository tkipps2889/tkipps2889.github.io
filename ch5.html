Von Neumann computer architecture, 4 major subsystems:

  1. Central Processing Unit (CPU)
  2. Memory
  3. Input devices
  4. Output devices

Von Neumann computer architecture, 3 characteristics:

  1. Stores data and instructions in memory
  2. Executes instructions sequentially one-by-one
  3. Allows programs to be stored in memory and modified

Four things the interface between the computer and the hardware does:

  1. Manages communication
  2. Allocates system resources
  3. Controls data transfer
  4. Provides entry points for software access
