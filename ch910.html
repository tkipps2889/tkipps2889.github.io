Term definitions:

  • Object Code: Compiled machine code that a computer can execute directly.
  • Source Code: The original code from which object code is compiled.
  • Procedural Language: High-level programming language that follows a procedure to execute code.
  • Integrated Development Environment (IDE): Software designed to provide all functions needed to develop software.
  • Software Development Lifecycle (SDLC): A six-stage proccess for developing software from planning to deployment.
  • Waterfall Model: A software development model focused on linear progression along a planned path.
  • Agile Model: A more flexible software development model focused on a cycle of small changes and testing, developing features as needed.

Term definitions part 2; SDLC:

  • Feasibility Study: A study to determine the viability of a project or component of a project.
  • Problem Specification: Complete definition of a problem that needs to be addressed.
  • Program Design: Planning out a program before coding begins.
  • Algorith Selection: The proccess of choosing which algorithm to use for a specific task.
  • Coding: The act of writing code for the program.
  • Debugging: Resolving issues and errors with the code.
  • Testing: Repeatedly running the program with different test cases to find issues that may have been missed.
  • Verification: The proccess of verifying that the program meets all requirements and specifications.
  • Documentation: Writing comprehensive documents explaining how the program and its code work.

Chapter 10: Tower of Babel: Programming Languages

Questions and Answers:

Q: Why does the textbook refer to "the Tower of Babel" regarding computer languages?
A: Because there are many different computer languages with thier own syntax and vocabulary, and code from one language is usually not understood by other languages.

Q: Why isn't there just one computer language?
A: Different languages were designed for different purposes and have different strengths or drawbacks.

Q: Why didn't FORTRAN use the less than symbol for comparing two numbers?
A: FORTRAN is an older language that uses syntax that would be considered unusual today

Q: What was COBOL designed for?
A: Buisness software (it stands for Common Buisness-Oriented Language, after all), particularly financial functions

Q: What was C designed for?
A: C was primarily designed for creating system-level software like drivers and operating systems.

Q: What is the difference between C and C++
A: C++ is an object-oriented language; it has classes and objects whereas C does not.

Q: Who or what is Ada named for and why?
A: It was named after Ada Lovelace, the first computer programmer.

Q: What is a java "applet"?
A: A small program written in java to perform a specific task, usually as part of a larger application.

Q: How much does Python cost to download, install, and use?
A: $0!

Q: What is Microsoft.NET?
A: An open-source software development platform created by Microsoft.

Q: What is SQL and what is it used for?
A: Structured Query Language is a data management language primarily used by databases.

Q: What is R and what is it used for?
A: R is another data-focused language used primarily for data analysis.

Q: What is Go and what is it used for?
A: Go is a general-purpose language designed to be easy to use, with a large library of common functions to make development more efficient

Q: What is Swift and what is it used for?
A: Swift is a language used for development of IOS applications.

Alternative programming paradigms:

  • Functional Programming: Function-centric paradigm that often stores functions in variables, uses them as arguments, etc.
  • Logic Programming: Paradigm that uses logical reasoning to adress a sequence of commands in order
  • Parallel Programming: Paradigm where multiple proccesses are carried out at the same time.
